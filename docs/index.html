Stock Tracker GUI

A desktop app for tracking stock trades & portfolio performance.

View on GitHub

Overview

Stock Tracker GUI is a Python desktop application that lets you log buy/short stock transactions, track current portfolio status, and calculate real-time profit/loss using live Yahoo Finance data.

Features

  • Record BUY and SHORT transactions.
  • View active positions with live prices.
  • Calculate total, monthly, and daily profits.
  • Store all data in a local SQLite database.

Installation & Usage

To run this application on your computer:

git clone https://github.com/yourusername/stock-tracker.git
cd stock-tracker
pip install -r requirements.txt
python app.py

Note: This is a desktop GUI app, so it won’t run in your browser. You must run it locally with Python installed.
